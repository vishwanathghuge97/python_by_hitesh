
**Tuple Unpacking:**

```python
tea_types = ("Black", "Green", "Oolong")

(black, green, Oolong) = tea_types

>>> black
# → 'Black'

```

Unpacking automatically extracts elements from a tuple and assigns them to individual variables in a single line. For this to work successfully, the number of variables on the left side must exactly match the number of items stored inside the tuple.
